What the Engineer Still Does

We don’t need a better coding agent alone. We need to automate the engineer role around today’s agents.

Workers · today

Capable agents already execute steps
  • Coding
  • Browser use
  • Deep research
  • Code review

They can act. They still don’t own the full job.

Engineer role · still mostly human

The runtime that turns workers into an outcome
  • Set the goal and quality bar
  • Allocate and prompt the right workers
  • Carry context and long-term memory
  • Evaluate independently and recover
  • Know when to stop — or escalate

Autonomy rises when the workflow role is automated — not when one worker gets smarter.
