# Beyond Loop Engineering

Shareable materials for **Beyond Loop Engineering** by Li Yin — creator of AdalFlow and founder at AdaL.

## Slides

After GitHub Pages deploys, open the repository’s Pages URL to view the interactive slide deck. Use the arrow buttons, ←/→, Space, Home, or End to navigate.

## Materials

- `index.html` — shareable interactive slide deck
- `materials/lesson.md` — lesson script and research notes
- `materials/demo_runbook.md` — demo companion
- `materials/illustrations/` — editable Excalidraw sources and rendered diagrams
